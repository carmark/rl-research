Rollout / Generation

  • 支持 vLLM、SGLang、TensorRT-LLM
  • v0.7 默认 Server 模式（动态批处理）
  • GRPO: chunk-dispatch-repeat 负载均衡
  • AgentLoop 支持多轮 agentic 任务
  • 异步生成: 2.35-2.67x 加速 (128 GPU)

Training / Optimization

  • FSDP / FSDP2 (ZeRO-3) + Megatron-LM
  • 5D 并行: DP + TP + PP + CP + EP
  • PPO / GRPO / DAPO / DPO / SFT
  • LoRA/PEFT 支持, 实验性 FP8 训练
  • 千亿参数级: DeepSeek-671B, Qwen3-235B

Data Processing

  • DataProto: 张量 + 元数据统一协议
  • Dispatch.DP_COMPUTE_PROTO 自动分片
  • TransferQueue (v0.7): 控制流/数据流解耦
  • 零拷贝传输, RDMA 支持
  • v0.8 将默认使用 TensorDict

I/O & Communication

  • Ray: 分布式 Actor 调度与编排
  • NCCL: 集合通信 (广播权重同步)
  • NIXL: P2P 点对点传输 (新增)
  • 3D-HybridEngine: 训练↔推理权重重分片
  • Checkpoint Engine: 统一权重同步抽象
